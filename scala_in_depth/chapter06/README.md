# Chapter 6 型別系統

“型別系統是 Scala 語言非常重要的組成部分。它使得編譯器能進行很多編譯時優化和約束，從而提高運行速度與避免程序錯誤。型別系統讓我們可以在我們自身周圍創建各種有用的**牆**，也就是所謂的**型別**。通過讓編譯器來跟蹤變量、方法與類的訊息，這些**牆**能幫助我們避免不小心寫出不正確的程式碼。你對 Scala 的型別系統所知越多，這能給編譯器更多的信息，讓型別的**牆**變得不那麼束縛，而同時仍然提供相同的保護。”
