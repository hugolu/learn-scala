# Regular Expression

參考連結: [規則表示式與提取器](http://openhome.cc/Gossip/Scala/RegularExpressionExtractor.html)

一些常用的範圍，可以使用預先定義的字元類別

| 字元 | 作用 | 說明 |
|------|------|--------|
| ```.``` | | 任一字元 |
| ```\d``` | ```[0-9]``` | 數字 |
| ```\D``` | ```[^0-9]``` | 非數字 |
| ```\s``` | ```[ \t\n\x0B\f\r]``` | 空白字元 |
| ```\S``` | ```[^ \t\n\x0B\f\r]``` | 非空白字元 |
| ```\w``` | ```[a-zA-Z_0-9]``` | 數字或是英文字 |
| ```\W``` | ```[^a-zA-Z_0-9]``` | 非數字與英文字 |

Greedy quantifiers

| 表示 | 說明 |
|------|------|
| ```X?``` | X出現一次或完全沒有 |
| ```X*``` | X出現零次或多次 |
| ```X+``` | X出現一次或多次 |
| ```X{n}``` | X出現n次 |
| ```X{n,}``` | X出現至少n次 |
| ```X{n,m}``` | X出現至少n次，但不超過m次 |
